## ⭐ GitHub Education Git Cheat Sheet (Best Quick Reference)

🔗 https://education.github.com/git-cheat-sheet-education.pdf  

**Why recommended:**
- Simple and beginner friendly  
- Covers most used Git commands  
- Very useful for quick revision

📌 Recommended for: **Beginners → Intermediate**

## ⭐ Atlassian Git Cheat Sheet (Industry Standard)

🔗 https://www.atlassian.com/git/tutorials/atlassian-git-cheatsheet  

**Why recommended:**
- Very detailed and professional  
- Covers Git workflows and common use cases  
- Great for developers working in teams

📌 Recommended for: **Intermediate → Advanced**

📌 Recommended for: **Beginners → Intermediate**

---

## ⭐ Oh Shit, Git!? (Real-Life Debugging Guide)

🔗 [https://ohshitgit.com/](https://ohshitgit.com/)

**Why recommended:**

* Focuses on *real mistakes developers make*
* Gives exact commands to fix Git disasters
* Extremely useful when you’re stuck (not just learning)
* Covers scenarios like:

  * Committed to wrong branch
  * Messed up rebase
  * Deleted something important
  * Broken history

📌 Recommended for: **Intermediate → Advanced (Must-have for real work)**

---

##  Suggested Usage

- Keep one cheat sheet open while practicing Git commands  
- Use cheat sheets for quick command lookup instead of Googling  
- Best for interview revision
